Block access to your admin pages by country.

== Description ==

Easy to use plugin, that blocks access to your wp-admin area by country.

== Installation ==

1) Install WordPress 3.5 or higher

2) Download the latest from:

http://wordpress.org/extend/plugins/tom-m8te 

http://wordpress.org/extend/plugins/a-forms

3) Login to WordPress admin, click on Plugins / Add New / Upload, then upload the zip file you just downloaded.

4) Activate the plugin.
